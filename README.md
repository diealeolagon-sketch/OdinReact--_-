# Odin Sena

Proyecto base creado con React, Vite, Tailwind CSS y React Router DOM.

## Requisitos

- Node.js instalado.
- npm instalado.

## Instalacion

```bash
npm install
```

## Ejecucion en desarrollo

```bash
npm run dev
```

## Scripts disponibles

```bash
npm run dev
npm run build
npm run preview
npm run lint
```

## Estructura principal

```text
src/
├── assets/
├── components/
│   ├── common/
│   └── layout/
├── pages/
├── styles/
│   └── index.css
├── App.jsx
└── main.jsx
```
